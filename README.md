# AI Stress Monitor

Real-time biometric stress monitoring dashboard built with React, Vite, TypeScript, Tailwind CSS, and shadcn/ui components.

## Local Development

Requirements:

- Node.js 18+
- npm

Install and run:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

Run tests:

```sh
npm run test
```

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
